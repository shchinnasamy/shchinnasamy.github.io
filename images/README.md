# Images Directory

This directory is intended for storing:
- Personal/professional photos
- Research project images
- Academic presentation screenshots
- Icons and graphical assets

All content has been cleaned from the original Jon Barron template.
Add your own images here.

Recommended image formats:
- JPG for photos and complex images
- PNG for logos, icons, and images with transparency
- MP4 for short videos/animations

Recommended naming convention:
- Use descriptive, lowercase names with underscores
- Example: `profile_photo.jpg`, `project_demo.mp4`
