# Academic Personal Website Template

This is a cleaned academic personal website template, originally based on Jon Barron's website structure (https://jonbarron.info/). The template has been cleaned of all personal content and is ready for customization.

## Features
- Static HTML website (no build process required)
- Responsive academic CV layout  
- GitHub Pages deployment ready
- Clean, professional academic styling
- Organized asset directories

## Usage
1. Add your personal/professional photo to `images/`
2. Add your academic papers and CV to `data/`
3. Edit `index.html` to include your information
4. Customize `stylesheet.css` if needed
5. Deploy via GitHub Pages

## Local Development
```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```
